1. Number of Values: =B4-B3+1

3. Probability of Value 1: =1/B5

4. Probability of Value 1 or Value 2: =1/B5+1/B5

5. Probability of Neither Value 1 and Value 2: =1-E5

6. Generate a Random number between 0 and 1: =RAND()
